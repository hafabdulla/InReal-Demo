# Advertising Reference

Use this for all paid advertising channels: Google Search Ads, Meta/Instagram Ads, LinkedIn Ads, display, retargeting, video ads, etc.

## The InReal Ad Register

**Tone:** Sharp, benefit-forward, like the best billboard you've ever read.
**Principle:** Every character counts. Every word must earn clicks.
**Goal:** Get the click. Let the landing page convert.

---

## Google Search Ads (SEM)

**Format:**
- Headline 1: 30 characters max
- Headline 2: 30 characters max
- Headline 3: 30 characters max (optional)
- Description 1: 90 characters max
- Description 2: 90 characters max (optional)

**Best practices:**
- Lead with the benefit in Headline 1
- Include a number if possible
- Include a CTA in one of the headlines or descriptions
- Match the landing page headline exactly (or very close)

**Example Ad Set (for "real estate investment $500"):**

```
Headline 1: Own Real Estate for $500
Headline 2: No Landlord Hassles. 7.8% Yield.
Headline 3: Start in 2 Minutes. Apply Today.

Description 1:
Fractional real estate in Bangkok, Dubai, Singapore.
Fully managed. Monthly rental income. BVI-regulated.

Description 2:
Join 1,200+ investors. $3.4M deployed.
Capital at risk. See our full disclosure. Apply now.
```

**Keywords to bid on (examples):**
- Real estate investing $500
- Fractional real estate
- Own apartment Bangkok
- Rent yield 7%
- Real estate portfolio app
- Cross-border property investment

## Meta / Instagram Ads (PPC)

**Format options:**
1. **Carousel** (5+ images, each with headline + copy)
2. **Single image** (1 image + primary text + headline + description)
3. **Video** (6-15 second video + text overlay)
4. **Collection** (featured item + related items)

**Character limits:**
- Primary text: 125 characters
- Headline: 27 characters
- Description: 30 characters
- CTA button: 17 characters

**Ad copy template (single image):**

```
PRIMARY TEXT:
"Own a piece of Dubai Marina 🏙️

$500 gets you started.
7.2% projected annual yield.
Zero property management.

Real estate for the rest of us."

HEADLINE:
"Real Estate for $500"

DESCRIPTION:
"Fractional property ownership"

CTA BUTTON:
"Invest Now"
```

**Video ad template (15 seconds):**

```
VISUAL SEQUENCE:
0–2 sec: Text overlay "Own real estate for $500" over city skyline B-roll
2–5 sec: Phone showing $500 investment
5–8 sec: Bar chart showing 7.8% yield
8–12 sec: Apartment interior
12–15 sec: CTA button overlay "Start Investing"

VOICEOVER:
"Real estate investing used to cost $300k.
Now it costs $500.
Same properties. Same returns. Zero management.
In 2 minutes, you could own a piece of global real estate.
[Link in bio]"
```

**Audience targeting:**
- Interest: Real estate investing, finance, wealth building
- Demographics: 25–55, household income $50k+
- Lookalike: Similar to existing investors
- Retargeting: Website visitors who didn't convert

**Context:** Someone visited your website but didn't convert.

**Sizes:** 300x250 (medium square), 728x90 (leaderboard), 160x600 (wide skyscraper)

**Copy template (retargeting someone who visited but didn't invest):**

```
HEADLINE:
"Still thinking about investing?"
```

## Video Ads (YouTube, Facebook, Instagram)

### 6-Second Bumper Ads

**Goal:** Create curiosity, not conversion (too short for CTAs).

```
VISUAL:
Fast cuts of global real estate (Dubai, Bangkok, Singapore)

TEXT OVERLAY:
"Real estate investing used to cost $300k.
Now it costs $500."

AUDIO:
Upbeat, professional music (no voiceover needed; text does the work)

CALL TO ACTION:
[Skip button] — if they don't skip, you won 6 seconds of attention
```

### 15-30 Second Video Ads

**Structure:**
- Seconds 0-2: Hook (surprising fact or contrast)
- Seconds 3-10: Explanation (2-3 key benefits)
- Seconds 11-15: Proof (stat or testimonial)
- Seconds 16-30: CTA (explicit next step)

```
SCRIPT (15-30 seconds):

[0-2 sec] TEXT/VO: "Your landlord invested. You didn't."
[Clips: Someone paying rent, then checking an account]

[3-8 sec] TEXT/VO: "Own real estate for $500. Zero landlord work."
[Clips: App demo, property interior, dashboard]

[9-12 sec] TEXT/VO: "Bangkok. Dubai. Singapore. 7.8% average yield."
[Clips: City skylines, rental income notification]

[13-15 sec] TEXT/VO: "Start in 2 minutes."
[Clips: Phone screen with investment button]

[CTA OVERLAY]: "Make Your First Investment"
```

## A/B Testing Framework

**Test one variable at a time:**

| Variable | Test A | Test B | Measure |
|----------|--------|--------|---------|
| **Headline** | "$500 Entry Point" | "Real Estate, No Landlord Work" | CTR, CPL |
| **Audience** | 25-40 age | 30-55 age | CPL, CPA |
| **Benefit** | Access (pillar 1) | Returns (pillar 4) | CTR, conversion rate |
| **Visual** | Property image | People investing | CTR, engagement |
| **CTA** | "Invest Now" | "Explore Properties" | CTR, conversion rate |

**Sample sizes:** Run for 2-4 weeks before declaring a winner. Need statistical significance (typically 100+ conversions per variant).

## Compliance Guardrails for Ad Copy

**Every ad mentioning returns or performance must include risk language. Options:**

### Short-form (most ads):
"Capital at risk. Projected returns are not guaranteed."

### Medium-form (if space allows):
"7.8% is a projected yield based on current rental data. Capital at risk. Past performance ≠ future results."

### Placement:
- Display ads: Small text in bottom corner
- Video ads: Text overlay in final 2 seconds
- Search ads: In description field
- Social ads: In the first comment (pinned)

**Red flags:**
- Any guarantee language ("will earn," "guaranteed," "risk-free")
- Misleading comparisons to savings accounts or stock returns
- FOMO language ("limited time," "act now," "exclusive")
- Unsubstantiated claims ("safe," "proven," "tested")

## Ad Copy Checklists

### Before Launch

- [ ] **Headline matches the product benefit:** Someone reading just the headline understands what they're buying.
- [ ] **Copy is specific:** Numbers, not adjectives. "7.8% yield" not "great returns."
- [ ] **CTA is action-specific:** "Explore properties," not "learn more."
- [ ] **Risk disclosure is present:** Any mention of returns includes appropriate risk language.
- [ ] **No weasel words:** "Potentially," "arguably," "up to" removed unless essential.
- [ ] **Landing page matches the ad:** The headline and benefit align exactly with what the landing page delivers.
- [ ] **Compliance approved:** Every claim is defensible with supporting documentation.

### After Launch (Weekly Check)

- [ ] **CTR is tracking:** Are people interested? (Benchmark: 1-2% is healthy for real estate)
- [ ] **CPL is reasonable:** (Benchmark: $2-10 CPL for qualified leads)
- [ ] **Conversion rate is acceptable:** (Benchmark: 5-15% of site visitors → registered users)
- [ ] **Cost per acquisition (CPA) is profitable:** (Benchmark: Lifetime value of investor should be 3-5x CPA)

## Common Ad Copy Mistakes to Avoid

| Mistake | Example | Fix |
|---------|---------|-----|
| Vague benefit | "Innovative real estate platform" | "Own Bangkok property for $500" |
| No numbers | "High yields" | "7.8% projected annual yield" |
| Feature focus | "AI-driven property matching" | "Find the right property in 2 minutes" |
| Weak CTA | "Learn more" | "See available properties" |
| Misleading claim | "Passive income opportunity" | "Earn rental income from real estate" |
| No social proof | Generic headline | "1,200+ investors. $3.4M deployed." |
| Ignored context | Same copy for all audiences | Different copy for first-timers vs. experienced investors |
